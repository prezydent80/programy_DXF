**Korzystanie z 'licznik_blokow' / 'licznik ramek'**
1. Zapisać kopie rysunku w folderze "liczenie blokow" jako .dxf, bardzo ważne żeby rysunek miał nazwe "rysunek.dxf"
2. Wejść w Wiersz Poleceń i wejść do folderu: "cd C:\Users\Użytkownik\Desktop\liczenie blokow"
3. Odpalić program: "python lb.py" / "python lr.py"
4. Jeżeli liczysz ramki, plik excela trzeba przenieść w inne miejsce lub usunąć, przed następnym użyciem programu
